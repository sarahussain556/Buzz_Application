**Technologies Used**


*Frontend*
-> Reactjs.
-> Material UI.
-> axios
-> timeago(to display time of feeds)

*Backend*
->Node js
->MongoDB
->Express
->Mongoose
->Postman(for testing)
->multer(for uploading files)

**Client can**
->Login
->Register
->post feed(photos,text,videos)
->like, unlike post
->follow/unfollow other users
->Logout

**to run backend**
->nodemon index.js

**to run Frontend**
->npm start
